# Smart Mini Vending Machine for Electrical Components

Microcontroller-based mini vending machine using Arduino Nano.
Features coin detection, product selection via button, DC motor control,
and LCD I2C display.

## Hardware
- Arduino Nano (ATmega328P)
- DRV8833 Motor Driver
- IR Obstacle Avoidance Sensor Module
- IR Optocoupler Speed Sensor
- PCF8574 I2C Button Expander
- 16x2 LCD I2C
- DC Motor 5V
